# ManifestFinder
Localiza e faz o download de arquivos utilizado pelo app Accela para fazer o download e inserção na Steam localmente.

Foi desenvol com o intuito de facilitar a utilização para aqueles que não desejam utilizar o metodo convencional do Accela que utiliza uma API onde o usuário tem que fazer logins diarios com sua conta do Discorde para permanecer utilizando.

Com o Manifest Finder o processo se torna muito mais simples!
